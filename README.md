# madre-server

### Project Stack

- Go(workspace)

### Prepare PostgreSQL Database

```shell
$ cd ./docker
$ docker compose up -d
```

### Start

```shell
# If air is installed
$ air

# If air is not installed
$ go run ./main.go
```
