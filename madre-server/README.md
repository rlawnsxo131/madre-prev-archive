# madre-server

### Project Stack
* MySQL
* Yarn Berry
* TypeScript
* Fastify
* Apollo Server
* TypeORM

### Prepare MySQL Database
```shell
$ cd ./docker
$ docker compose up -d
```

### Start
```shell
$ yarn start:dev
```
